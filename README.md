# To-Do-List
A simple and useful to-do list application. Users can add tasks, mark them as complete when finished, and delete them if they wish. Since all tasks are stored on localStorage, they do not get lost even if the page is refreshed.

# Features
✅ Adding a new task
✅ Marking tasks as completed
✅ Deleting tasks
✅ LocalStorage support
✅ Adding tasks using the Enter key on the keyboard
✅ Responsive and simple interface
